Q: How does the hybrid retriever rank results?

A:
`hybrid.HybridRetriever` ranks by fusing BM25, vector, and graph signals into one ranked list (`hybrid.HybridRetriever (retrieval\hybrid.py:55-183)`).

The merge logic uses Reciprocal Rank Fusion over multiple ranked node-id lists: `hybrid._rrf_merge (retrieval\hybrid.py:46-52)`. The retriever then returns the top-k results from `hybrid.HybridRetriever.retrieve (retrieval\hybrid.py:74-130)` as `hybrid.HybridResult (retrieval\hybrid.py:34-43)`.
